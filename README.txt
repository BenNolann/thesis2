## run_fastqc.py

Runs FastQC on a directory containing single-end fastq raw read files.

Usage: `python run_fastqc.py -h` for more information.

FastQC: `https://www.bioinformatics.babraham.ac.uk/projects/fastqc/`

## trimmed_seqs.py

Runs trimmomatic on raw read files. To be used in conjunction with run_fastqc.py.

Usage: `python trimmed_seqs.py` for more information.
